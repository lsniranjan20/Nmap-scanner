# Nmap Scanner Tool 

A simple Python-based Nmap scanning tool created for learning Cyber Security and network scanning concepts.

##  Features
- Host discovery
- Port scanning
- Service detection
- Beginner-friendly output

##  Tools Used
- Python
- Nmap
- Kali Linux

##  How to Run
1. Install nmap:
   sudo apt install nmap

2. Run the tool:
   python3 nmap_scanner.py

##  Purpose
This project is created for educational purposes to understand:
- Network scanning
- Reconnaissance
- Basic penetration testing workflow
